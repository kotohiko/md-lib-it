## 1.6   连接

### 1.6.1   TCP/IP

TCP 提供了：

-   无差别的数据传输；
-   按序传输（数据总是会按照发送的顺序到达）；
-   未分段的数据量（可以在任意时刻以任意尺寸将数据发送出去）。
